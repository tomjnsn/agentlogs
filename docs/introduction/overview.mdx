# What is AgentLogs?

AgentLogs captures and analyzes transcripts from AI coding assistants, giving you visibility into how these tools work on your codebase. Track costs, understand behavior patterns, and maintain a searchable history of AI interactions.

## Supported Agents

AgentLogs currently supports the following AI coding assistants:

| Feature         | Claude Code | OpenCode |
| --------------- | ----------- | -------- |
| Transcripts     | ✓           | ✓        |
| Auto-sync       | ✓           | ✓        |
| Commit tracking | ✓           | —        |
